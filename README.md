# python-ml-setup

First project in my 8-week ML learning plan. This repo contains my
development environment setup and notebooks for basic PyTorch tensor
operations.

## Setup

This project uses [uv](https://docs.astral.sh/uv/) for Python and
dependency management.

```bash
git clone https://github.com/chavoshh/python-ml-setup.git
cd python-ml-setup
uv sync
```
